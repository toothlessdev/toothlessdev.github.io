# 자동화 테스트 포스트

이 포스트는 VitePress 블로그의 자동화 기능을 테스트하기 위해 만들어졌습니다.

## 테스트 내용

- ✅ 사이드바 자동 업데이트
- ✅ 포스트 목록 자동 업데이트
- ✅ 카드 레이아웃 자동 적용
- ✅ 카테고리별 필터링

## 기술 스택

- VitePress
- Vue 3
- TypeScript
- Node.js

이 파일을 저장하면 자동으로 사이드바와 블로그 목록에 나타나야 합니다!
